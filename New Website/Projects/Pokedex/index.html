Pokedex

The Original Pokedex

In June 2016, I embarked on my first ever real journey into development. At the age of 12, I decided to make a real life Pokedex, powered by a Raspberry Pi.

It began as a text only program which simply told you a Pokemon's name after entering in their ID. I bought a keypad, a few buttons and LEDs, and 2 small screens, to connect to the Raspberry Pi. You could enter a Pokemon ID with the keypad and then it would display a picture of that Pokemon on the big screen, with some information on the small screen.

I also decided to buy a Raspberry Pi camera, which I used to take a photo and then overlay a chosen Pokemon on top of the picture. It would then email this picture to me.

We were then able to go to a local maker space and get a case we designed laser cut.

The code for this project can be found here: https://github.com/Flye001/Pokedex

Pokedex V2

The next year, I wanted to re-make this project with the knowledge I had gained over that year. I decided to make the new version run on a single touchscreen in a much smaller, cleaner case. This time, the case was 3D printed, and it just had a single touchscreen display, with a camera mounted in the back.

The UI was completely reworked to accommodate the touchscreen, and I was again pleased with the end result. It had all the same features as the first Pokedex, however this time, much smaller and portable.
