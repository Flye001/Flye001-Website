FlyeChat

FlyeChat is a web-based, real time chat client, similar to Discord.

History Of FlyeChat

I originally started developing it at the start of 2021 in January, using Python with a client and server application, connected using the socketserver library. It worked quite well, even integrating Discord Rich Presence, however there was one major flaw: whenever I converted it to an exe for other people to try, Windows Defender would flag it as a virus.

It was this that ultimately drove me to make a web version. I firstly made a Python web version using Flask, however my HTML/CSS skills were not very good. It was at this point Conor joined the project. Ever since, he has been responsible for making the great designs for FlyeChat. The Flask web version worked fairly well, but I felt limited by Python and wanted to try something new.

At this point, I decided to learn Microsoft's ASP.NET. I remade the whole system from scratch using ASP.NET and the SignalR, and it turned out pretty well.

FlyeChat V3

During late 2021, I once again decided to recreate FlyeChat with the knowledge I had gained over the last year. This version was once again created with ASP.NET, but had a completely new design, once again thanks to Conor.

This version included voice chats, screen sharing, servers, friends, and much more.
